Changes from Rev 1

Summary of Rev 2 Changes

  • Power input consolidated to a single 2-position screw terminal (one for 24VAC L1, one for 24VAC L2)
  • Separate fuses on both L1 and L2 for circuit power; additional 1A fuse on L1 to relay contact commons
  • Zone common return (J_COM) and 24VAC L2 input are the same electrical point — one terminal serves both
  • 8-position screw terminal replaces individual zone connectors
  • DIP switch added — 8-position, raised type, labeled "Zones" — for manual zone selection
  • Rotary binary switch added — 4-position, labeled "Time" — selects run time (15 / 30 / 45 / 60 min)
  • Manual Start button added — illuminated momentary push button, labeled "Man. Start"
  • PCB extended 25 mm in terminal direction for case divider clearance
  • Plastic enclosure specified — split design, top over electronics, bottom over terminals
  • BOM retitled "Suggested BOM"

1. System Overview

The Sprinkler-8 is an 8-zone 24VAC irrigation relay controller based on the ESP32-WROOM-32E. It switches eight solenoid zones using relay contacts powered by the 24VAC supply. Rev 2 adds stand-alone manual operation controls (DIP zone select, time rotary switch, start button) directly on the board so the controller can operate without a WiFi connection or mobile device.

Supply

  • 24VAC, standard irrigation transformer
  • Single 2-position input terminal (L1 + L2)
  • L1 and L2 individually fused

Outputs

  • 8 zone outputs via relay NO contacts
  • All relay contact commons fed from 24VAC L1 through a dedicated 1A fuse
  • Zone device common return connects to 24VAC L2 (same terminal as L2 power input)

Manual Controls (new)

  • 8-position raised DIP switch — zone select ("Zones")
  • 4-position rotary binary switch — time select ("Time")
  • Illuminated push button — manual start ("Man. Start")

Enclosure

  • Plastic split case
  • Top section: encloses board electronics
  • Bottom section: exposes terminal strip and user controls
  • PCB extended 25 mm to provide clearance for internal case divider

3. Power Architecture

3.1 Input Terminal

There is one 2-position screw terminal (J_PWR) for the 24VAC supply. One position is 24VAC L1, the other is 24VAC L2. No barrel jack. No power-select jumper.

TerminalLabelConnection
J_PWR pin 124VAC L1Fuse F1 (circuit power) and Fuse F3 (zone contact commons)
J_PWR pin 224VAC L2Fuse F2 (circuit power return) — same electrical point as J_COM (zone device return)

3.2 Fusing

Both AC legs are independently fused. Do not share a single fuse across L1 and L2.
RefRatingPosition in CircuitProtects
F11A24VAC L1 → rectifier bridge D1Electronic circuit power (ESP32, relay coils, 5V/3.3V supply)
F21A24VAC L2 → rectifier bridge D1Electronic circuit power return leg
F31A24VAC L1 → relay contact common busZone output power — protects against wiring faults at solenoid side; independent of F1

F1 and F3 both source from 24VAC L1 but are independent fuses. A solenoid wiring fault that blows F3 does not kill the electronics (F1 stays intact, ESP32 keeps running).

3.4 Common Return — J1 / J2 Same Point

In Rev 1, the zone device common return (J2) and the 24VAC L2 input were separate terminals. In Rev 2 they are the same electrical point — 24VAC L2 enters at J_PWR pin 2 and that net also connects directly to J_COM (the zone common return terminal). Only one physical terminal is needed. Label it COM / L2 on the silkscreen.

4. Zone Output Circuit

4.1 Relay Contact Wiring

NetDescription
ZONE_PWR_BUS24VAC L1 → F3 (1A) → relay K1–K8 contact COM pins (all 8 relay commons bussed together)
ZONE_1 … ZONE_8Relay K1–K8 NO contacts → J_ZONES terminals 1–8
ZONE_COM_RETJ_COM terminal (24VAC L2) — external device solenoid returns connect here
Solenoid wiring: Each irrigation solenoid connects between one ZONE_n terminal and the ZONE_COM_RET (J_COM) terminal. When the relay closes, 24VAC flows through the solenoid and returns via the common.

4.2 Zone Output Terminal

One 8-position 3.5 mm screw terminal block (J_ZONES) with positions labeled Z1–Z8. Placed at the edge of the terminal zone, accessible from outside the case.

4.3 MOV Snubbers

Retain one 47V MOV disc (RV1–RV8) across each relay NO contact and COM. Solenoid inductive kick protection.

7. PCB Requirements

7.1 Board Extension for Case Divider

Critical dimension: The terminal zone must extend 25 mm beyond the edge of the electronics zone. This 25 mm clearance is required to fit the internal plastic case divider wall between the two zones without interfering with any components.

Place all screw terminals (J_PWR, J_ZONES, J_COM), SW_DIP, SW_ROT, and SW_START within the terminal zone. No active electronics or relay bodies should be within 5 mm of the divider boundary on either side.

7.2 Zone Boundary

Mark the electronics / terminal divider boundary clearly on the silkscreen layer with a dashed line. This line should align with where the physical case divider wall will sit.

7.3 Layer Stack and General Rules

  • 2-layer PCB minimum. 4-layer preferred if routing density requires.
  • ESP32 antenna keep-out: 15 mm clear zone on all layers around module antenna end.
  • Relay switching loop (LM2596 buck) kept physically tight — no signal traces in that region.
  • GPIO12 must have a 10 kΩ pull-down resistor to GND (strapping pin — see Rev 1 spec).
  • ULN2803A COM pin (pin 10) must connect to +5V.
  • Terminals J_ZONES and J_COM placed at one long edge so wire routing exits cleanly from the case.

8. Enclosure / Case

8.1 Split Design

The enclosure is a two-part plastic snap or screw-together shell:

SectionCoversAccess
Top / Electronics half Relays, ESP32, power supply, UART — all active electronics Closed — no user access required during normal operation. USB-C port may have a cut-out or remain internal.
Bottom / Terminal half Screw terminals (J_PWR, J_ZONES, J_COM), DIP switch, rotary switch, push button Open front or open bottom — terminals must be accessible for wire connection. Controls (DIP, rotary, button) must be operable without removing the case.

8.2 Divider Wall

An internal wall integral to the case sits between the electronics and terminal zones. This wall:

  • Provides a physical barrier between mains-derived voltages at the terminals and the low-voltage electronics
  • Aligns with the 25 mm extended PCB boundary (see Section 7.1)
  • Has slots or notches for the wiring that crosses between zones (relay contact wiring to terminals)

8.3 Material and Finish

  • ABS or PC plastic, flame-retardant grade (UL94 V-0 preferred)
  • Light grey or off-white — no specific colour requirement but avoid dark colours that make terminal labeling hard to read
  • DIN rail mounting clip on the base is desirable but not required
